## Optimization Tips

1. Compress PNGs using tools like ImageOptim or TinyPNG
2. Consider WebP format for better compression (with PNG fallback)
3. Max width: 1200px for full-width screenshots
4. Keep total directory size under 5MB if possible

## Usage in Markdown
